This project is an Arduino-based gas detection system that uses the MQ-3 sensor to monitor the presence of gases such as ethanol, carbon monoxide (CO), and carbon dioxide (CO₂). The system reads analog values from the sensor, estimates gas concentration levels, and displays the readings on an I2C LCD. When gas levels exceed predefined safety thresholds, a buzzer is activated to alert users, and a warning message is shown on the display. The project also outputs detailed gas readings to the serial monitor for analysis. It is built using Arduino (C++), with components including an MQ-3 gas sensor, buzzer, and LCD display, making it suitable for safety monitoring applications.
